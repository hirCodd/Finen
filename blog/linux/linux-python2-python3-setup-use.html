linux(ubuntu)中不同版本的python的安装与切换

Finen python2.7python3.x 2018-03-17 1000000

# Linux下不同版本的Python安装与切换

# 方法一

# 安装第三方源

sudo add-apt-repository ppa:jonathonf/python-3.6

# 安装更新并安装python3.x

sudo apt-get update
sudo apt-get install python3.6

# 检查版本

update-alternatives --list python

# 建立版本链接

update-alternatives --install /usr/bin/python python /usr/bin/python2.7 1
update-alternatives --install /usr/bin/python python /usr/bin/python3.5 2
update-alternatives --install /usr/bin/python python /usr/bin/python3.6 3

# 选择你的版本

sudo update-alternatives --config python

# 输入数字选择你的版本即可

# 其他问题----------看下面

E: Could not get lock /var/lib/dpkg/lock - open (11: Resource temporarily unavailable)
E: Unable to lock the administration directory (/var/lib/dpkg/), is another process using it?

出现类型这样的问题咋办？=====>

sudo rm /var/cache/apt/archives/lock
sudo rm /var/lib/dpkg/lock

> 但是这样不能解决问题：可能还会出现问题，解决方案：

sudo dpkg --configure -a
sudo apt-get update

# 方法二

此方法可能不能解决有些问题，例如(NodeJS,webpack等问题需要python2，python3，因为我在开发中遇到需要python2，python3的问题，需要随时切换)

# 下载python3.6.x版本

下载地址：Python3.6.x版本下载

# 安装

cd python3.6.x  //打开python目录
./configure --prefix /usr/local/python3.6   //安装python到指定目录
sudo make
sudo make install

# 设置

增加软链：

update-alternatives --install /usr/local/python3.6 python /usr/bin/python3.6 3

# 选择相对应的版本

sudo update-alternatives --config python
